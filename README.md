# Bread Pudding

Loosely based of @jpweller's boilerplate.

## How To: 
```
$bundle install ('$gem install bundler' if not installed yet)
```
```
npm install
```
```
grunt sync (for browser sync)
```
```
grunt watch
```

## Known Issues
